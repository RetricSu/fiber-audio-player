## Test Coverage
### Health API
- GET /healthz returns status 200 with ok=true

### Podcasts API (Public)
- GET /api/podcasts lists all podcasts
- Returns empty array when no podcasts exist

### Podcasts API (Admin)
- POST /admin/podcasts creates podcasts with validation
- GET /admin/podcasts lists all podcasts (authenticated)
- GET /admin/podcasts/:id gets specific podcast
- PUT /admin/podcasts/:id updates podcast fields
- DELETE /admin/podcasts/:id deletes podcast
- All endpoints require valid API key authentication

### Episodes API (Public)
- GET /api/podcasts/:id/episodes lists published episodes
- GET /api/episodes/:id gets specific episode
- Returns 404 for non-existent resources

### Episodes API (Admin)
- POST /admin/episodes creates episode metadata
- GET /admin/episodes lists episodes for podcast
- GET /admin/episodes/:id gets episode details
- PUT /admin/episodes/:id updates episode (draft only)
- DELETE /admin/episodes/:id deletes episode
- POST /admin/episodes/:id/publish publishes ready episodes
