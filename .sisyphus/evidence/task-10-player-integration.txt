## Key Integration Points

1. **Episode Data Flow:**
   - PodcastList → onPodcastSelect → selectedPodcast state
   - EpisodeList → onEpisodeSelect → selectedEpisode state
   - selectedEpisode → toAudioPlayerEpisode() → AudioPlayer.episode prop

2. **Payment Flow:**
   - AudioPlayer.play() → payment.start(episode.id, chunkSeconds)
   - payment.start() → createSession(episodeId) → Backend API

3. **Layout:**
   - Left sidebar (1/3 width): Podcast/Episode browser
   - Right main area (2/3 width): AudioPlayer + NodeStatus + PaymentHistory
